Dynamic MPI Machinefile Maker Released (v1.0.0)!

Feb 9th, 2002

I’ve had requests from users to integrate ganglia into their MPI installations. As a first step I’m releasing a Perl script which creates a dynamic load-balanced MPI machinefile. Any nodes in your cluster that are down are not included in the list and the number of CPUs for each node is listed as well. To get this great script, go to the download page
